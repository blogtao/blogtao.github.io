前言

由于某些需要，我们需要获取外网 IP ，下面是个人收集的获取 公网 IP 的一些网站。如果你需要更好的服务或需要调整排列顺序，欢迎联系我。

IP 服务

自动获取 IP

  • www.myip.com/

  • ifconfig.me/

  • www.ipify.org/

  • ip.cn/

  • www.cip.cc/

  • www.ipip.net/

  • api.getmyip.me 支持 HTTP 和 HTTPS 访问，可查看是否使用透明代理服务。

  • ip.sb/

  • ifconfig.co/

  • members.3322.org/dyndns/getip，由国内某个云厂商提供。

  • ip.taobao.com/ipSearch.html

查询 IP 信息

将 $IP 替换成所要查询信息的 IP 地址

  • ip-api.com/json/$ip

  • ip.taobao.com/service/getIpInfo.php?ip=$IP，可以通过 ip.taobao.com/service/getIpInfo.php?ip=myip 进行测试。
