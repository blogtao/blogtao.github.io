解决 Apple ID (苹果账号)被停用方法

前言

近期不知什么原因导致 Apple ID 账号被停用，当在 Mac 上 App Store 登录 Apple ID 更新和下载应用时，弹窗大概提示

您的 Apple ID 已被停用，如需帮助，请联系 iTunes 支持，https://www.apple.com/support/itunes/ww/ 。

重设密码

当访问弹窗的链接，选择地区国家，进入联系 Apple 支持，选择 Apple ID 后，发现刚好有一个已被停用 Apple ID问题类别，点进去后，官方推荐通过 重设你的密码这个选项，安装提示步骤进行操作即可。

电话技术支持

被动联系

若通过重设密码无法解决(重设密码对我不起作用)，可试着通过电话与苹果支持取得联系，也就是点击重设你的密码后面的立即与 Apple 支持部门人员通话。选项，填写相关资料后，等待苹果支持与你取得联系。

不一会儿，我收到02161791000来电，当时向来电技术人员(MM)说明了情况，中途进行了转接，转接后的客服(GG)询问了一些个人和账号等信息（eg：有没有满 18 岁等），当向我索要账号安全问题验证时，立刻警惕了起来，怀疑接到一个假的 Apple 客户电话（因为可以通过账号安全问题可以重置你的密码等操作，你将你的安全问题透露出去，而后不进行更改的话，就相当于告诉别人你的账号密码，这是个隐藏的炸弹。）。感觉奇怪，于是在浏览器中输入来电电话，进行搜索，一些与隐私泄露的话题出现在眼前。

于是假意向客服说已经忘了安全问题答案(三个安全问题答案已经告诉他一个，后悔啊)，当问及还有其他解决办法时，客服回复无其他办法。无奈挂断电话。

注：个人认为，联系客服是让其解决遇到的问题，而不是收集用户的隐私信息的。

主动联系

最后的希望！在上次电话转接前，客服(MM)告诉我一个电话，如果在转接过程中挂断，可拨打400-666-8800再次进行解决。抱有一丝侥幸心理试了一下，问题一下子就解决啦！

这次通话是两个MM接的，第一个MM询问我是否下载后助手之类的app，我回复没有，这个账号只在Mac上登录，没有在手机上登录，说了好几遍，好像我理解错了MM说的问题，而后又转接到了第二个MM电话中，这次我直接告诉 客服MM 安全问题已经忘记，还有其他解决办法吗？她说有的于是有个下面的步骤。

AppleCare 技术顾问已发送验证代码以确认身份，这个验证码显示在设备上


设备验证码

在 https://appleid.apple.com 上登录Apple ID，获取PIN Code，提供给客服


登录Apple ID点击获取支持PIN

生成 PIN


点击生成PIN码

大家会注意到临时支持 PIN 码是有时间限制的，个人感觉这比把安全问题告知他人强。（网络电话一般都会录音的(eg:10086)，不知苹果的客服电话有没有录音哈）


获取PIN码，发给客服

按照客服指示完成操作之后，再 App Store 点击下载或更新会弹出一个协议条款 点击同意之后，Apple ID能够恢复正常使用。

注：写这篇杂记的目的，在于帮助与我遇到相同问题的人，同时也希望大家加强安全意识。
