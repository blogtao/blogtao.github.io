前言

总所众知，如果想让更多的用户，在苹果设备上使用自己的 App 就得需要专门的会员账号，目前会员账号主要分为 3 大类，4 种账号类型：

  • 个人（Individual）
  • 组织（Organization）
    • 公司（Company / Organization）
    • 企业（Enterprise / Organization）
  • 教育机构（Educational Institutions）

苹果对账号的介绍 Compare Memberships

今天我们来介绍一下 苹果开发者账号 续费流程和一些需要注意的事情。

续费前你需要准备的资料
1、双币卡（银联+V/M 标识的卡片）
2、发票抬头（如需报销）

通知续费

一般提前一个月，会弹出续费弹窗；点击“Renew Membership”，会跳转到“送货详情 - 安全结账”网页。

送货详情

在“送货详情 - 安全结账”页面，填写联系方式和发票相关信息。

注意：

  • 联系方式中的电子邮箱建议填写目前账号拥有者(主账号)的邮箱，这样避免续费后账号信息发生变更。
  • 发票可以选择个人或企业发票抬头，但苹果方面目前开出的发票为普通发票，苹果方面给出了以下信息：

根据中国有关税务规定，只有公司银行转账支付才能开具增值税专用发票。你购买的产品只接收国际卡支付，因此我们不能向你开具增值税专用发票。特此告知

付款方式

这里主要填写信用卡支付的一些相关信息（账号、安全码、日期、账单人姓名、账单地址等），鉴于安全性，暂不放图。

查看订单

填写完信用卡信息之后，会进入“查看订单 - 安全结账”界面，这个页面中有上几个页面填写的一些信息，如果订单信息没问题的话，就可以 点击 “立即下单”，进行付款下单啦。


付款完成

“立即下单”付款完成之后，会跳转到此页面。

订单列表

如果我们不放心的话，可以前往订单列表，查看已订购产品

订单详情


官方给出的订单帮助电话：400-666-8800

开发者中心，过期时间进行变更

FQA

订单失败情况

  • 使用 vpn 或 代理

  • 信用卡问题

    信用卡问题

  • 信用卡需要是双币种“美元+人民币” 且具有“银联+Visa”或者“银联+Mastercard”标志，而非全币种。
  • 信用卡余额足够支付
  • 信用卡账号、安全码、日期、账单人姓名、账单地址，填写与你申请信用卡填写一致
