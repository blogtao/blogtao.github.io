前言

最近申请一些服务需要用到 域名证书，从事互联网的人士，或多或少的都能接触到 SSL 证书，也就是 https ,浏览器中有个小锁的标识。今天由我来给大家讲一下 域名证书 ，域名证书 与 SSl 证书 是两种不同的概念。

域名证书是证明您是该域名的拥有者，并已在域名数据库中备案，可由域名注册商颁布；而 SSL 证书是证明网站可靠性的标识，可从不同的签名机构获得。

如何获取域名证书

想要获取域名证书，首先要知道域名现所在的域名注册商（eg：阿里云、新网、Godaddy等），向域名注册商进行索要。

以 namesilo 域名注册商为例，登录账号点击想要获取证书的域名，看到 Print Certificate:，点击后面的 Print 按钮，可获取电子版的域名证书。主要包括以下内容：域名、拥有者、过期时间、域名注册商；
