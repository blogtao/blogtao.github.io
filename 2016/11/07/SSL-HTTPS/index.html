引言

近一年来，HTTPS 在国内很火的，这离不开免费证书的颁发机构，其次也离不开开发者对知识的探索，同时也离不开用户对网络安全的重视，下面我介绍一下关于 HTTPS。

什么是 HTTPS

HTTPS（Secure Hypertext Transfer Protocol）安全超文本传输协议是在 HTTP（Hypertext Transfer Protocol）超文本传输协议基础上开发，基于安全套接字层（SSL）进行安全数据传输的加密版Http协议。简单讲，HTTPS = HTTP + SSL，是 HTTP 的安全版。

HTTPS 的优势

防流量劫持

全站 HTTPS 是根治运营商、中间人流量劫持的解决方案，不仅可以杜绝网页中被插入的小广告，更可以保护用户隐私安全。

提高搜索排名

采用 HTTPS 可以帮忙搜索排名的提升，提高站点的可信度和品牌形象。

杜绝钓鱼网站

HTTPS 地址栏绿色图标可以帮助用户识别出钓鱼网站，保障用户和企业的利益不受损害，增强用户信任。

HTTPS 与 HTTP 的区别

HTTP（Hypertext Transfer Protocol）超文本传输协议是用来在 Internet 上传送超文本的传送协议，它可以使浏览器更加高效，使网络传输减少。但 HTTP 协议采用明文传输信息，存在信息窃听、信息篡改和信息劫持的风险。
HTTPS(Secure Hypertext Transfer Protocol) 安全超文本传输协议是一个安全的通信通道，它基于 HTTP 开发，用于在客户计算机和服务器之间交换信息。HTTPS 使用安全套接字层(SSL)进行信息交换，简单来说 HTTPS 是 HTTP 的安全版，是使用 TLS/SSL 加密的 HTTP 协议。

什么是 SSL 证书

SSL证书是数字证书的一种，遵守 SSL协议，由受信任的数字证书颁发机构CA（如：Symantec）在验证服务器身份后颁发，具有服务器身份验证和数据传输加密功能。

SSL 证书问世

SSL 证书就是遵守 SSL 安全套接层协议的服务器数字证书。而 SSL 安全协议最初是由美国网景 Netscape Communication 公司设计开发的,全称为:安全套接层协议 (Secure Sockets Layer) , 它指定了在应用程序协议(如 HTTP、Telnet、FTP)和 TCP/IP 之间提供数据安全性 分层的机制,它是在传输通信协议(TCP/IP)上实现的一种安全协议,采用公开密钥技术,它为 TCP/IP 连接提供数据加密、服务器认证、消息完整性以及可选的客户机认证。由于此协议很 好地解决了互联网明文传输的不安全问题,很快得到了业界的支持,并已经成为国际标准。SSL 证书由浏览器中“受信任的根证书颁发机构”在验证服务器身份后颁发,具有网站身份 验证和加密传输双重功能。

SSL 证书作用

服务器部署了SSL证书后可以确保从用户电脑到服务器之间的传输链路上是高强度加密传输的，用户在浏览器上输入的机密信息和从服务器上查询的信息是不可能被非法篡改和窃取的。同时向网站访问者证明了服务器的真实身份，此真实身份是通过第三方权威机构验证的。

颁发 SSL 机构

国内厂商经常合作的机构，从国内厂商申请的证书，一般都是与国外机构进行合作的。

国外

0、赛门铁克 (Symantec) 是 SSL/TLS 证书的领先提供商，为全球一百多万台网络服务器提供安全防护。选择赛门铁克后，证书颁发机构 (CA) 将妥善保护您的网站和信誉，让您安枕无忧。
1、GeoTrust 是全球第二大数字证书颁发机构，也是身份认证和信任服务的领导者
2、Let’s Encrypt，3个月续签一次，建议使用自动续签功能。
3、StartCom 免费，不建议使用

国内

0、亚洲诚信® 是Symantec™（赛门铁克）的白金合作伙伴，登录入口，近期(截止到20161130)可以关注微信，使用兑换码免费使用一年。
1、腾讯证书申请，是基于亚洲诚信的，可以免费申请使用，有效期也是一年，百度、阿里云也提供相关 SSL 证书。
2、中国权威数字证书认证机构自主研发，纯国产证书，是国际CA浏览器联盟组织成员，同时通过国际WebTrust 认证，遵循全球统一鉴证标准；提供 7 * 24 小时金融级的安全保障服务；完善的风险承保计划。中文版 CPS（全球信任体系电子认证业务规则）便于用户理解双方权利和义务。(注: CFCA 服务器证书目前不支持Safari浏览器)

SSL 证书分类

用于网站HTTPS化的SSL数字证书，当前主要分为DV SSL、OV SSL、EV SSL三种类型的证书。
DV 证书是域名级别验证的证书，通常无需人工审核，快速办法。适合小型网站。
