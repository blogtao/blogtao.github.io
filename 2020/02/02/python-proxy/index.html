前言

由于某些原因（保护隐私），不希望对方看到自己的请求IP等信息，我们可以借助 IP 代理，让代理服务器作为中间人，从而保护我们的隐私。下面我们将以 Python 中 requests 网络库为中心展开。

使用

目前有两种主流的设置代理方法：一个是 http 协议，一个是 socks 协议。
