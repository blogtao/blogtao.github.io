前言

远程控制 macOS 桌面，不像控制 Windows 桌面设置那么简单。由于 Apple 注重安全性，所以需要系统授权，下面以 向日葵 远程桌面控制服务软件进行举例说明。

无论是 macOS 控制 macOS 桌面, 还是 Windows 控制 macOS 桌面。无论是在内网控制，还是在外网控制。大多数都是通过 App 实现的，在被控制的 macOS 上授予该 App 以下权限。

权限
